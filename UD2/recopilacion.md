
# Recoplicación de información técnica

| Herramienta / Componente | Función |
| :--- | :--- |
| **pfSense** | Firewall y enrutador para la seguridad perimetral y gestión del tráfico de la red. |
| **DMZ (VLAN 10)** | Zona Desmilitarizada; subred aislada que expone servicios a Internet protegiendo la red interna. |
| **INTERNA (VLAN 20)** | Red de área local (LAN) privada y segura donde operan los usuarios y recursos confidenciales. |
| **MONITOREO (VLAN 30)** | Segmento o entorno dedicado a las herramientas de supervisión del estado y seguridad de la infraestructura. |
| **VPN** | Red Privada Virtual (Virtual Private Network) para permitir el acceso remoto cifrado a la red corporativa. |
| **Portainer** | Interfaz gráfica (GUI) para la gestión y administración simplificada de contenedores Docker o Kubernetes. |
| **Nginx Proxy Manager** | Gestor de proxy inverso para redirigir el tráfico web entrante y administrar certificados SSL. |
| **Active Directory** | Servicio centralizado de Microsoft para la gestión de identidades, usuarios, permisos y políticas de grupo. |
| **Wazuh** | Plataforma de seguridad (SIEM/XDR) para la monitorización, detección de intrusos y gestión de registros (logs). |
| **PRTG** | Herramienta para la monitorización del rendimiento de la red, ancho de banda y disponibilidad de dispositivos. |
| **Honeypot** | Sistema señuelo configurado para atraer a posibles atacantes, analizar sus tácticas y alertar sobre intrusiones. |
| **AWS S3** | Servicio de almacenamiento en la nube altamente escalable, comúnmente usado para respaldos (backups) o almacenamiento de datos. |
| **API / Claude** | Integración con inteligencia artificial para la automatización de tareas, análisis de logs o asistencia interactiva. |
| **Estaciones de trabajo** | Equipos finales (endpoints o computadoras) utilizados por los empleados para realizar sus tareas diarias. |
